Detailed Description

template<typename T>
class bitpit::ProxyVector< T >

Metafunction for generating a list of elements that can be either stored in an external vectror or, if the elements are constant, inside the container itself.

Usage: Use ProxyVector<Type> to declare a list of elements that can be either stored in an external vectror or, if the elements are constant, inside the container itself. When the ProxyVector is destroyed, the elements of the list will be destroyed only if owned by the ProxyVector o bject itself.
